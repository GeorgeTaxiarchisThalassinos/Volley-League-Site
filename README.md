# VOLLEY LEAGUE ΑΝΔΡΩΝ 2026 

Ένα πλήρως λειτουργικό και responsive website για το πρωτάθλημα Volley League Ανδρών 2026. Η ιστοσελίδα δημιουργήθηκε στο πλαίσιο πανεπιστημιακής εργασίας (Σχεδίαση Εφαρμογών & Υπηρεσιών) με σκοπό την εξάσκηση σε HTML5, CSS3, και Responsive Web Design.

##  Χαρακτηριστικά (Features)

* **Responsive Design:** Σχεδιασμός Desktop-First με προσαρμοσμένο σύστημα στηλών (12-column grid system) και χρήση Media Queries για άψογη προβολή σε Tablets και Κινητά.
* **Προφίλ Ομάδων:** Αναλυτικές κάρτες ομάδων με λογότυπο, ρόστερ, προπονητικό επιτελείο, βίντεο στιγμιοτύπων και επίσημους συνδέσμους.
* **Πρόγραμμα & Βαθμολογία:** Οργανωμένη παρουσίαση αγώνων και βαθμολογίας. Περιλαμβάνει δυνατότητα οριζόντιου scroll (overflow-x) στις μικρές οθόνες για να μην "σπάει" το layout.
* **Φόρμες (Authentication & Data Entry):** * Καθαρό UI (user interface) για Είσοδο (Login) και Εγγραφή (Sign-up).
    * Εξειδικευμένες φόρμες προσθήκης Αποτελέσματος και Ομάδας. 
    * *Καινοτομία:* Στα κινητά, ο πίνακας του ρόστερ μετατρέπεται αυτόματα σε μορφή καρτών με CSS (χρήση `:nth-child::before`), για τέλεια εμπειρία χρήστη (UX) χωρίς οριζόντιο scroll.

##  Τεχνολογίες που χρησιμοποιήθηκαν

* **HTML5:** Σημασιολογική δόμηση (Semantic HTML) με χρήση tags όπως `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<figure>`.
* **CSS3:** * **Flexbox:** Για την ευθυγράμμιση των καρτών και του μενού πλοήγησης.
    * **Custom Grid System:** Δημιουργία δομής με `.row` και `.col-` (col-1 έως col-12) για αρθρωτό (modular) σχεδιασμό.
    * **Media Queries:** 2 Breakpoints (1024px για Tablets, 768px για Mobile).
    * **Ψευδοστοιχεία:** `::before` για δυναμική εισαγωγή κειμένου στα κινητά.

##  Δομή Αρχείων

Το project είναι οργανωμένο με καθαρή αρχιτεκτονική φακέλων:

* `/html/` - Περιέχει όλα τα αρχεία της δομής (`index.html`, `clubs.html`, `matches.html`, `table.html`, `login.html`, κλπ).
* `/styles/` - Περιέχει τα αρχεία μορφοποίησης. Το `global.css` ελέγχει τον κεντρικό κορμό και το Grid, ενώ υπάρχουν ξεχωριστά αρχεία για κάθε σελίδα (π.χ. `clubs.css`, `auth.css`) για ευκολότερη συντήρηση του κώδικα.
* `/media/` - Εικόνες ομάδων, λογότυπα και βίντεο που χρησιμοποιούνται στο site.

##  Πώς να τρέξετε το project

1. Κατεβάστε τα αρχεία ή κάντε clone το repository.
2. Ανοίξτε τον φάκελο στον υπολογιστή σας.
3. Κάντε διπλό κλικ στο αρχείο `html/index.html` για να το ανοίξετε σε οποιονδήποτε σύγχρονο browser (Google Chrome, Firefox, Safari κλπ). 

*Σημείωση: Δεν απαιτείται τοπικός server καθώς το project βασίζεται σε στατικά αρχεία (HTML/CSS).*
